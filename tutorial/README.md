   <img width="80" alt="コードエリア" src="https://github.com/oomori-kun/toiodo_maz/assets/17058514/1c0fc6de-b729-4d49-8b4c-67b9a739f4b3" >

   * 【注意】「スタート位置....」のブロックが実行されると、toioはマップのスタート位置まで移動し、指定された向き（上、下、右、左）に向きます。
